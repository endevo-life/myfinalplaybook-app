# Q12 Mobile Team Brief

## Summary
Convert the current Q12 web assessment into a React Native app that works on iPhone and Android, keeps the scoring engine, and replaces the PDF-first experience with a daily action flow.

## Main product shift
- Web quiz becomes mobile app.
- PDF becomes optional backup.
- Daily action screen becomes the core habit loop.
- Push notifications and tracking become part of the product.

## Estimated effort
- MVP: 3 to 6 weeks.
- Store-ready version: 6 to 10 weeks.
- Polished version with deeper automation: 10 to 14+ weeks.

## Recommended direction
- Build with Expo + React Native.
- Reuse the current Q12 logic.
- Keep GHL for CRM and automation.
- Track app events from day one.
- Keep the first release simple and cost-friendly.

## Team decision points
- Mobile-first daily action experience vs PDF delivery.
- Whether to use login or email-only access.
- Which events should be tracked.
- Which reminders should be automated.
- Whether the PDF should exist only as an export.
